# Real-Time Vital Signs Monitoring System 🏥
> 🏆 1st Prize Winner — College Project Competition 2026
> 📋 Patent Filed — Innovative IoT Healthcare Solution
An IoT healthcare device that monitors Heart Rate, Temperature, 
SpO2, Weight, and Respiration Rate with cloud connectivity 
and automated audio alerts.
## 📁 Project Files
| File | Description |
|------|-------------|
| `heart_rate_monitor.ino` | MAX30105 SpO2/IR sensor → cloud |
| `vital_signs_main.ino` | Weight + Temp + BPM + MP3 alerts → cloud |
## 🔧 Hardware Used
- ESP8266 WiFi Module
- MAX30105 Pulse Oximeter & Heart Rate Sensor
- HX711 Load Cell Amplifier (Weight)
- DS18B20 Temperature Sensor
- DFPlayer Mini MP3 Module
- Analog Pulse Sensor
## ✨ Features
- Real-time weight, temperature, BPM monitoring
- SpO2 and IR-based heart rate measurement
- Automated audio alerts via MP3 player
- Cloud data posting via HTTP POST
- Remote audio track command from cloud (JSON)
- Alert repeats every 10 seconds automatically
## ☁️ Cloud Connectivity
- POST: Sends weight, temperature, BPM to IoT cloud
- GET: Receives remote audio track commands in JSON format
